O que são os números binários?

Os número binários são uma reprsentação alternativa um algoritmo utilizando somente dois digítos, no caso, o 0 e 1. Podemos escrever diferentes quantidades usando combinações destes símbolos em diferentes posições. Sendo assim, se você escreve uma combinação somente com 0 e 1, você automaticamente já está fazendo uma representação em código binário.

Como converter os números binários em número decimal?

Para converter os números binários em números decimais é um assunto bastante complicado, porque eu dei uma breve pesquisada e realmente é complicado de se calcular. Para converter um número binário para o número decimal equivalente basta multiplicar cada dígito pela potência de 2 relativa à posição por ele ocupada e somar os resultados. Assim por exemplo o número binário 101 equivale ao número 5 no sistema decimal.
Exemplo: 101 = 1*22 + 0*21 + 1*20 = 1*4 + 0*2 + 1*1 = 4 + 0 + 1 = 5

E como converter número decimal em binário?

Para que você consiga fazer a conversão do decimal para o binário, você precisa fazer divisão consecutiva do 2 com o número que você deseja. Quando você terminar de dividir o número, ou o número vai ser divisivel por completo por 2, e caso esse seja o caso, o primeiro algoritmo do número vai ser 0, ai deois você vai fazer novamente a divisão com 2, só que dessa vez com o quociente da divisão do primeiro número, e caso o número não seja divisivel por 2, o algoritmo será 1.
